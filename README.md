APIを活用した要約アプリ



②課題内容（どんな作品か）

APIを利用したテキスト要約ツールです。
入力された文章をAIが要約し、情報整理や内容把握を効率化することを目的としています。



③アプリのデプロイURL

https://mitsuko117.github.io/Summary/



④アプリのログイン用IDまたはPassword（ある場合）

なし



⑤工夫した点・こだわった点

Gemini または ChatGPT のAPIを使った要約機能の実装に挑戦しました。
エラーが頻発しても途中で諦めず、最終的に Gemini API と接続できるところまで実装した！



⑥難しかった点・次回トライしたいこと（又は機能）

API連携部分が最も難しく、エラーメッセージの原因特定に時間がかかりました。
API連携を組み込んだ自分のプロダクトのデモを、より完成度高く作成したい。



⑦フリー項目（感想、シェアしたいこと等なんでも）

最初は2〜3時間ほどエラーが出続け、ChatGPTにも繰り返し相談しましたが解決できず…
最終的には Google の公式ドキュメントを確認しながら調整したところ、10分ほどで解決しました。
公式ドキュメントほんまに大事。
